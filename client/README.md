## 클라이언트 폴더 (Vite + React, JavaScript)

이 폴더는 Vite 기반의 React(JavaScript) 프로젝트입니다.
현재 메인 화면은 컴포넌트(`AppBar`, `CategoryTabs`, `ItemCard`, `ItemCardList`) 중심으로 구성되어 있습니다.

### 1. 의존성 설치

```bash
cd "client"
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 표시되는 주소(기본값 `http://localhost:5173`)로 접속하면
재고 카드 기반 메인 화면을 확인할 수 있습니다.
